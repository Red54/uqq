uqq
===

QQ for ubuntu phone, using webqq 3.0 protocol

为ubuntu phone设计的一个QQ客户端，使用Qt/QML语言实现，使用webqq3.0协议。目前仅初步设计出UI，及操作方式，可进行demo展示。
通讯协议部分未实现，并且未进行任何优化。
